# infinite-craft-bot

A Infinite craft (https://neal.fun/infinite-craft/) bot that i made a LONG time ago, with some combinaisons I found.
You can freely use the 22124 combinaisons I found to make anything you want. 
If you use the bot, please make a Pull request to update the list so anyone can enjoy our work.
The algorithm is random it's not searching for a specific thing.

Format of `proxies.json`:
```json
[
  "http://1.2.3.4:5678",
	"http://7.2.3.4:566",
	"http://1.2.3.4:49",
	"http://111.222.123.213:645"
]
```
